Javascript tiene tipos

Los tipos vienen definidos por el valor y no por la variable

  • object
  • null
  • undefined
  • number
  • string
  • boolean
  • Symbol
  • typeof {} === 'object'
  • typeof null === 'object'
  • typeof undefined === 'undefined'
  • typeof 42 === 'number'
  • typeof "42" === 'string'
  • typeof false === 'boolean'
  • typeof symbol() === 'symbol'
  • Más:
    • typeof function(){} = "function"
    • typeof [] = "object"
